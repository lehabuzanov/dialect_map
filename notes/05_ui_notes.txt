1. Левая панель
- Поиск работает по населённым пунктам, районам, вопросам и вариантам ответа.
- Переключатели атласов показывают два набора вопросов: ЛАРНГ и ДАРЯ.
- Выпадающий список и карточки ниже показывают только вопросы выбранного атласа.
- Одновременно можно выбрать один или два вопроса.
- Кнопка сброса очищает выбранные вопросы, район и населённый пункт.

2. Карта
- Круглые маркеры показывают населённые пункты, загруженные в рабочую таблицу.
- Контуры районов помогают навигации и районной сводке.
- Цветные границы соседних регионов показывают внешний контекст карты.
- Ареалы показывают зону распространения выбранного вопроса или варианта ответа.
- Изоглоссы показываются только при одновременном выборе двух разных вопросов.

3. Логика выбора
- Если выбран один вопрос, на карте остаются только те пункты, где по нему есть ответ.
- Если выбраны два вопроса, на карте остаются только те пункты, где есть ответ хотя бы на один из них.
- При выборе одного вопроса пункты окрашиваются по вариантам ответа.
- Если в одном пункте зафиксировано несколько ответов, маркер становится многоцветным.
- При выборе населённого пункта открывается карточка пункта со списком вопросов и ответов.
- При выборе района открывается карточка района со сводкой по числу пунктов, наблюдений и вопросов.
- При наведении на пункт показывается ответ или ответы по выбранному вопросу.

4. Условные обозначения
- Синий маркер: обычный населённый пункт без активного выделения.
- Выделенный маркер: выбранный населённый пункт.
- Многоцветный маркер: в пункте записано несколько ответов на активный вопрос.
- Цветной ареал: зона распространения выбранного ответа или вопроса.
- Пунктирная линия: изоглосса между двумя одновременно выбранными вопросами.
- Цветной пунктирный контур: граница соседнего региона.
