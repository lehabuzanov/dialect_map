1. Боковая панель слева
- Поиск по населённому пункту, району, признаку и языковой единице.
- Фильтр по разделам: Все, Гласные, Согласные, Морфология.
- Список признаков с возможностью выбора одного или двух признаков.
- Карточка выбранного пункта, района или признака.
- Легенда условных обозначений.

2. Карта
- Точки населённых пунктов показываются круглыми маркерами.
- Районы показываются тонким контуром и выделяются при выборе.
- Ареалы показываются полупрозрачной заливкой поверх карты.
- Изоглоссы показываются только при одновременном выборе двух разных признаков.

3. Принцип работы
- Если выбран один признак, на карте показываются связанные с ним точки и ареалы.
- Если выбраны два признака, дополнительно показываются изоглоссы между этой парой признаков.
- При клике по населённому пункту открывается карточка пункта и список наблюдений в нём.
- При клике по району открывается краткая сводка по району.
- При клике по признаку открывается карточка признака, примеры и связанные населённые пункты.

4. Условные обозначения
- Синий маркер: обычный населённый пункт без активного выделения.
- Выделенный маркер: выбранный населённый пункт.
- Цветной ареал: зона распространения выбранного признака.
- Пунктирная линия: изоглосса между двумя одновременно выбранными признаками.
- Полупрозрачные ландшафтные зоны: вспомогательный природный фон.
