1. Левая панель
- Поиск выполняется по населённому пункту, району, вопросу и вариантам ответа.
- Переключатели атласов показывают два набора вопросов: ЛАРНГ и ДАРЯ.
- Выпадающий список и карточки ниже показывают только вопросы выбранного атласа.
- Одновременно можно выбрать один или два вопроса.
- Кнопка сброса очищает выбранные вопросы, район и населённый пункт.

2. Карта
- Круглые маркеры показывают все населённые пункты, загруженные в рабочую таблицу.
- Контуры районов используются для навигации и краткой районной сводки.
- Ареалы показывают зону распространения выбранного вопроса.
- Изоглоссы показываются только при одновременном выборе двух разных вопросов.

3. Логика выбора
- Если выбран один вопрос, на карте подсвечиваются связанные с ним пункты и его ареал.
- Если выбраны два вопроса, каждому вопросу назначается свой цвет, а на карте дополнительно появляются изоглоссы между ними.
- Населённые пункты без наблюдений по выбранному вопросу остаются на карте как фоновая сетка.
- При выборе населённого пункта открывается карточка пункта со списком вопросов и ответов.
- При выборе района открывается карточка района со сводкой по числу пунктов, наблюдений и вопросов.
- При выборе вопроса открывается карточка вопроса с вариантами ответа, примерами и связанными населёнными пунктами.

4. Условные обозначения
- Синий маркер: обычный населённый пункт без активного выделения.
- Выделенный маркер: выбранный населённый пункт.
- Цветной ареал: зона распространения выбранного вопроса.
- Пунктирная линия: изоглосса между двумя одновременно выбранными вопросами.
